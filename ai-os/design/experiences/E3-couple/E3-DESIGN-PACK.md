## Stitch Prompts

### PROMPT 1 — Dashboard Below-Fold + Wedding Day Mode

```
Design the couple dashboard below-fold sections and the Wedding Day mode for "רגע לפני" — a Hebrew-first (RTL) wedding management SaaS.

CONTEXT: The couple checks this dashboard 3–7x per week for 16 weeks. The hero (dark luxury, countdown, readiness) is already built and approved. Design the scrollable content area below the hero and the special Wedding Day Mode screen.

DIRECTION: Warm Romantic for below-fold content. Dark luxury (matching hero) for Wedding Day Mode.

SCREENS TO DESIGN:
1. Below-fold dashboard (planning mode, 4-5 sections)
2. Wedding Day Mode (full-screen, date = today)

BELOW-FOLD SECTIONS (in order):
Section 1 — "הפעולה הבאה שלכם" (Next Action):
  Gold outlined card (not filled)
  Single most important next action, detected by stage
  One action button: warm gold link to the relevant page
  Examples: "שבצו 23 אורחים" / "שלחו הזמנות ל-45 ממתינים" / "אשרו את הקייטרינג"

Section 2 — "מי מגיע?" (RSVP Summary):
  3 stat pills: confirmed / declined / pending (gold background, cream text for confirmed)
  Gold progress bar: confirmed/total
  Encouragement text if >70%: "רוב האורחים אישרו — כל הכבוד!"

Section 3 — "מה עוד נשאר?" (Alerts):
  2–4 alert cards, assistant-style language (not clinical):
  ⚠️ Amber cards for attention items (soft amber bg, dark text)
  ✅ Green cards for completed milestones
  Never use red/error styling — warmth always
  Hebrew, encouraging language

Section 4 — Daily Inspiration:
  Full-width quote card (cream background, gold top border)
  Frank Ruhl Libre italic for quote text
  "——" divider + quote source (if any)

Section 5 — "גישה מהירה" (Quick Access):
  2×2 grid of access cards: Guests / Checklist / Vendors / Budget
  Each: icon (48px) + label + chevron

WEDDING DAY MODE (daysLeft === 0):
Full-screen dark background (#1C1008)
Gentle animated heart icon at top (subtle pulse, gold)
"היום הגדול הגיע" — Frank Ruhl Libre 900, white, very large
"חתונת [CoupleName]" — Heebo 400, gold subtitle
Date — muted white

If event timeline exists — show as vertical list:
  Time (Heebo 600, gold) + event label (Heebo 400, white)
  Clean, spacious, no decorative chrome

Primary CTA: "נווטו לאולם" — full-width, gold, 56px
4 secondary action cards in 2×2: Contacts / Seating / Gallery / Messages

PALETTE: Below-fold: ivory #FDFAF5, gold #C5A46D, dark #1C1008
         Wedding Day: dark #1C1008 background, white + gold text
FONTS: Frank Ruhl Libre (emotional headers) + Heebo (data, labels)
RTL: always. Mobile 390px primary.
EMOTIONS: Below-fold: "I'm organized. I'm making progress. This is my home."
          Wedding Day: "It's real. Today is the day. I'm ready."
```
